Математичні Операції

  • Додавання (+): Операція об'єднання двох чисел або виразів для отримання їх суми

    a + b = ?

  • Віднімання (-): Операція видалення одного числа або виразу з іншого для отримання різниці

    a - b = ?

  • Множення (× або *): Операція повтореного додавання числа до себе або іншого числа певну кількість разів

    a * b = ?

  • Ділення (÷ або /): Операція розподілу числа на рівні частини або розподілу одного числа на інше

    a / b = ?

  • Піднесення до степеня (^ або **): Операція піднесення числа до певного степеня

    a ^ b = ?

  • Взяття кореня (√): Операція знаходження числа, яке піднесене до певного степеня дає дане число

    √a = ?

  • Модуль (| |): Операція повернення абсолютного значення числа, відстані числа від нуля на числовій прямій

    |a| = ?

  • Остача від ділення (%): Операція повернення залишку від ділення одного числа на інше

    a % b = ?

  • Синус угла (sin): Операція повернення значення синуса кута

    sin a = ?

  • Тангенс угла (tan): Операція повернення відношення протилежної сторони трикутника до його прилеглої катети

    tan a = ?

  • Котангенс угла (ctg): Операція повернення оберненого значення тангенсу, тобто ctg = 1/tan

    ctg a = ?

  • Арксинус угла (asin): Операція оберненої функції синуса, яка повертає кут, що має задане значення синуса

    asin a = ?

  • Аркосинус угла (acos): Операція оберненої функції косинуса, що повертає кут, що має задане значення косинуса

    asin a = ?

  • Арктангенс угла (atan): Операція оберненої функції тангенса, що повертає кут, який має задане значення тангенсу

    asin a = ?

  • Аркотангенс угла (actg): Операція оберненої функції котангенсу, яка повертає кут, що має задане значення котангенсу

    asin a = ?

  • Натуральний логаріфм (log): Операція, яка визначає, до якого ступеня потрібно підняти певне число, щоб отримати дане значення

    asin a = ?
